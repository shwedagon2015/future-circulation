## プロジェクト画像構成 (Asset Configuration)

本プロジェクトで使用している主要なイメージ画像は、以下の通り構成・配置されています。

| 配置セクション | 画像ファイル名 | 内容説明 |
| :--- | :--- | :--- |
| **ABOUT** | `green_beans.jpg` | ミャンマー産コーヒー生豆のクローズアップ |
| **OUR COFFEE** | `coffee_cherry.jpg` | 収穫された完熟コーヒーチェリーの様子 |
| **CIRCULATION** | `coffee_processing.jpg` | 現地の農園における選別作業風景 |

### 画像の更新方法
画像ファイルはGitHubリポジトリのルート直下に配置されており、`index.html` 内にてRAWリンク（`https://raw.githubusercontent.com/shwedagon2015/future-circulation/main/画像名`）を参照しています。新しい画像を差し替える際は、ファイルをアップロードした後に上記URLが正しく読み込めているか確認してください。
