# Future Circulation - Myanmar Coffee & Culture Project

ミャンマー山岳地域のコーヒーと日常文化を届けるブランド「Future Circulation」の公式サイトです。

## 概念
一杯のコーヒーから、未来の循環を。
私たちは、単なるコーヒー販売ではなく、その背景にある文化やストーリーを現代のギフトとして再編集し、生産者と消費者が持続可能な形でつながる仕組みを目指しています。

## 使用技術
このWebサイトは、モダンなWeb技術を組み合わせて制作されています。

- **HTML5**: マークアップ
- **Tailwind CSS**: レスポンシブで洗練されたスタイリング
- **Three.js**: 背景のパーティクルエフェクトによる空間演出
- **GSAP (GreenSock Animation Platform)**: スクロールアニメーションとフェードイン効果
- **Lenis**: スムーススクロールの実装

## プロジェクトの目的
- ミャンマーの高品質なコーヒーと伝統文化の普及
- 持続可能な直接取引（ダイレクトトレード）の啓蒙
- ストーリーテリングを通じたブランド体験の提供
